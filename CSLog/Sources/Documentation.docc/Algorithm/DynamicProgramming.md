# Dynamic Programming

## Overview

문제를 작은 부분으로 나누어서 해결 후 그 결과를 저장해 나중에 다시 사용하는 기법이다. 

중복되는 문제가 등장하고, 작은 부분 문제의 합을 모아 큰 문제를 해결할 수 있을 때 적합하다.
예를 들어서 피보나치 수열과 [Merge Sort](<doc:Sort>) 가 있다. 

또한 어떤 정보를 순차적으로 접근해서 해결할 때도 유용하다. index를 0부터 n까지 순차적으로 접근하는 상황이 대표적이다. 이런 경우 Bottom up으로 해결하는데 Tabulation이라고 표현한다.
